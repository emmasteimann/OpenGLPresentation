Implementation: Rectangle

To make a rectangle we have to add three more vertices, as well as indices explaining to the GPU how to iterate over the vertices to make two triangles. 

      const static GLVertex vertices[] = {
        {{1, -1, 0}}, // Vertex 0
        {{1, 1, 0}}, // Vertex 1
        {{-1, 1, 0}}, // Vertex 2
        {{-1, -1, 0}} // Vertex 3
      };
    
      const static GLubyte indices[] = {
      0, 1, 2,
      2, 3, 0
      };
